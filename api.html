Authentication

All API requests must include an API key in the Authorization header. You can generate API keys from your dashboard settings.

Authorization: Bearer YOUR_API_KEY

Base URL: https://api.aiautomate.com/v1

Workflows

GET /workflows

List all workflows

POST /workflows

Create a new workflow

{ "name": "My Workflow", "description": "Automate email notifications", "trigger": { "type": "webhook", "config": {} }, "actions": [...] }
GET /workflows/{id}

Get a specific workflow

PUT /workflows/{id}

Update a workflow

DELETE /workflows/{id}

Delete a workflow

Tasks

GET /tasks

List all tasks

GET /tasks/{id}

Get task details

Webhooks

Webhooks allow you to receive real-time notifications when events occur in your account.

POST /webhooks

Register a webhook

{ "url": "https://your-app.com/webhook", "events": ["workflow.completed", "task.failed"], "secret": "your_webhook_secret" }
